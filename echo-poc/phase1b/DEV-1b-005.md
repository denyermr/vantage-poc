# DEV-1b-005 — G2 Set D exemption pending Phase 1c trunk-layer build

**Date:** 2026-04-19
**Gate:** G2 (MIMICS Implementation Gate, published_table arm, Set D)
**Session:** Phase 1b, Session E, Phase E-1
**Sibling DEV entries:** DEV-1b-004 (gradient-arm dielectric-configuration amendment), DEV-1b-006 (v0.2 physics promotion)
**Cross-references:** Decision Memo 2; SPEC.md §4, §13 G2 gate definition; `phase1b/physics/g2_anchor_spec.md` v0.2 §Set D → v0.3 `status: EXEMPT`; Vantage Green Paper v4.1.0 §8 (Phase 1c NISAR L-band validation).

## Spec said

`SPEC.md` §4 and `phase1b/physics/g2_anchor_spec.md` v0.2 §Set D specify a four-row anchor (D.1–D.4) against McDonald et al. 1990 Fig. 10 MIMICS curves for a walnut orchard at L-band 1.25 GHz, θ ∈ {40°, 55°}, polarisations {HH, VV}:

> `use_trunk_layer` must be set True for Set D; all other sets must leave it False. … All four rows (D.1–D.4) pass individually.

The `use_trunk_layer=True` code path requires structural extensions beyond the Toure single-crown adaptation that is the Phase 1b production configuration (SPEC §4):

- Four canopy branch classes (trunk branches, primary, secondary, stems — M90 Table I, p.482) rather than the single `N_b_per_m3` scalar.
- A cos⁶(θ) zenith orientation PDF for trunk branches (M90 Table I) in addition to the Gaussian zenith PDF used for the Toure single crown.
- Complex vegetation dielectric (M90 Table III, p.486: Im(ε) = 2.5–11.3 across tissue classes). The v0.1 real-only Ulaby & El-Rayes proxy in `reference_toure.py` / `mimics.py` cannot represent this.
- L-band 1.25 GHz wavelength support at the canopy-scattering level. Frequency is already parameterised in `params.freq_hz`, but the vegetation dielectric and the real-only UEL proxy are implicitly C-band-tuned; L-band would require dual-dispersion UEL.

## What was done

At the first G2 run (2026-04-18, end of Session D), all four Set D rows raised `NotImplementedError` because `mimics_toure_single_crown(params, use_trunk_layer=True)` raises with a clear message directing the caller to Session E. The harness recorded them as `status: UNIMPLEMENTED` in `outputs/g2_equivalence_result.json`.

Session E — after reviewing Decision Memo 2 — does **not** implement the trunk-layer code path. The scope of the trunk-layer implementation (four branch classes, cos⁶ PDF, complex vegetation dielectric with Im(ε), L-band support, and a parallel `MimicsMcDonaldParamsTorch` configuration path) is multi-session-class effort and is structurally orthogonal to the Phase 1b production configuration (Moor House heather, C-band, crown-only, peat). Combining it with the Session E P2 physics promotion (UMF finite-cylinder form factors — see DEV-1b-006) risks the fallback clause (NaN gradients, non-finite σ°, >5× training wall-clock regression). G2 sign-off cannot happen over an unstable physics module underneath it; better to narrow the Session E scope and ship a clean G2 close.

Set D is **exempted** from G2 pass/fail scoring in Session E. The exemption is **pre-registered as deferral, not cancellation**: Set D becomes a **hard prerequisite for Phase 1c** (NISAR L-band comparative validation, Green Paper v4.1.0 §8). When the trunk-layer code path is built for Phase 1c — where the L-band + trunk-layer + complex-dielectric configuration is the production operating regime — the Set D rows reactivate as a Phase 1c entry gate.

Code changes:

1. `phase1b/physics/mimics.py::mimics_toure_single_crown` — the `NotImplementedError` message when `use_trunk_layer=True` is updated to cite DEV-1b-005 and Phase 1c rather than pointing at Session E.

2. `phase1b/physics/equivalence_check.py::_run_set_D` — each Set D row returns with `status: EXEMPT` (was `status: UNIMPLEMENTED`), a DEV-1b-005 cross-reference in the `note` field, and does **not** count against the published_table arm pass/fail. Exempt rows are reported separately in the result JSON under a new `n_exempt` counter.

3. `phase1b/physics/equivalence_check.py::run_published_table_arm` — exempt rows are excluded from the arm pass-predicate. Active Set D row count is 0; Set A (4) + Set B (4) + Set C (5 active + the dropped C.3 self-pass = 6) + Set C2 (24 secondary from Table 11-1, see DEV-1b-006) constitute the arm.

4. `phase1b/physics/g2_anchor_spec.md` v0.2 → v0.3 — §Set D header re-marked with `status: EXEMPT`, cross-referenced to DEV-1b-005 and Phase 1c. Anchor values (D.1 = −11.73 dB, D.2 = −11.35 dB, D.3 = −12.17 dB, D.4 = −12.17 dB) retained verbatim; `phase1b/refs/anchor_reads/anchor_reads_v1.json` §set_D is unchanged. When Phase 1c reactivates Set D, the exemption is lifted with a companion DEV entry (likely DEV-1c-00N at that time).

5. `phase1b/deviation_log.md` — summary row added.

## Reason

Scope discipline at the Session E boundary. The Decision Memo 2 recommendation ("Option 4 — diagnostic-led targeted promotion") called for promoting approximations the first-run output implicates and exempting anchors whose production regime is orthogonal to the Phase 1b production configuration. Set D tests a canopy configuration (walnut-orchard trunk layer + L-band + complex dielectric) whose binding physics (two dominant branch classes in the wavelength-comparable regime; strong double-bounce) are **not** the binding physics of Moor House heather at C-band. The exemption costs nothing to the Phase 1b controlled comparison — the physics chain at Moor House's production regime is covered by Sets A, B, C (wheat, C-band, crown-only), the numpy_port arm (Moor House / sparse-canopy operating envelope), and Set E (wheat reference with Dobson mineral per DEV-1b-004).

Tying the exemption to Phase 1c rather than leaving it open-ended preserves the pre-registration integrity: Set D is not deleted, it is held pending a specific future gate that requires the same physics extensions to exist anyway. A Phase 1c without a trunk layer cannot run NISAR L-band, so the trunk-layer build is on the Phase 1c critical path regardless — Set D re-instatement is a by-product of that build, not additional overhead.

## Gate impact

**G2, published_table arm.** Set D rows no longer count against pass/fail; the arm is evaluated over Sets A, B, C (and Set C2 per DEV-1b-006). The arm-level pass predicate is tightened accordingly.

**G1 / G3 / G4:** no impact. G1 covers Phase 1 baselines; G3 covers Oh validity at Moor House peat s = 1–5 cm; G4 is the Dobson vs Mironov diagnostic at Moor House. None of these involve the trunk layer or L-band.

## Resolution

Amendment to `g2_anchor_spec.md` v0.3. Code changes in `phase1b/physics/mimics.py` (error message) and `phase1b/physics/equivalence_check.py` (Set D EXEMPT handling) per above. No tests are added for the trunk-layer path in Session E (it does not exist); a placeholder test in `tests/unit/test_mimics_torch.py` confirms `use_trunk_layer=True` still raises `NotImplementedError` and that the exception message cross-references DEV-1b-005.

When Phase 1c begins, a companion DEV entry (DEV-1c-00N or equivalent) lifts the exemption, the trunk-layer code path is implemented with its own specification (parallel to SPEC §4 for the walnut-orchard / L-band production regime), and Set D's four rows run as a Phase 1c entry gate alongside whatever new Phase 1c-specific anchors are pre-registered at that time.

## Impact on pre-registered criteria

**None for Phase 1b.**

- No tolerance loosening: Set D tolerance (±0.5 dB) is unchanged; the rows are simply not scored in the Session E G2 verdict.
- No anchor re-read: M90 Fig. 10 anchor values (D.1 = −11.73, D.2 = −11.35, D.3 = −12.17, D.4 = −12.17) are unchanged in `anchor_reads_v1.json` and `g2_anchor_spec.md` v0.3.
- No weakening of Phase 1b's scientific claim: the Phase 1b controlled comparison's success or failure rests on Moor House heather C-band performance vs the Phase 1 baselines, not on walnut-orchard L-band forward-model fidelity.
- No change to outcome categories, secondary criteria, or the Phase 1b diagnostic decision tree (SPEC §10, §11).

## Re-instatement criteria

Set D returns as a scored gate when **all** of the following are true:

1. Phase 1c trunk-layer code path built in `phase1b/physics/mimics.py` (or a Phase 1c-specific `phase1c/` module) with all four structural extensions above.
2. Complex vegetation dielectric with Im(ε) available (dual-dispersion Ulaby & El-Rayes or equivalent).
3. L-band canopy-scattering path validated against an independent anchor (either re-reading M90 Table V, which is a new anchor, or a comparable published L-band MIMICS tabulation).
4. A companion DEV entry (DEV-1c-00N) documents the lift and re-scores Set D.

Until all four are met, Set D's exemption stands.
