## Trigger

Phase E-1 diagnostic run at Session E close. The PyTorch mechanism-decomposition helper `mimics_toure_single_crown_breakdown_torch` (P3 deliverable) was probed at T94 Set A θ=30° CHH wheat inputs under both the SPEC §6 primary peat-Mironov dielectric (current default) and the Dobson-mineral dielectric that DEV-1b-004 authorised for rows E.1 / E.2. Three concrete observations:

- **Bare-soil σ°_oh:** peat-Mironov **−55.95 dB** (ε clamped to 1.01 by DEV-007 floor) → Dobson-mineral **−9.23 dB** (anchor −11 dB, Δ = 1.8 dB). The clamp is crushing the soil contribution at T94's wheat m_v = 0.17 g/cm³.
- **Set A.2 total:** peat **−20.63 dB** (canopy-dominated, flat across θ — a signature of ground-path suppression, not canopy saturation) → Dobson-mineral **−6.73 dB** (anchor −10.44, Δ = 3.71 dB).
- **Set C.1 direct-ground mechanism:** peat **−64.63 dB** (mechanism fully suppressed by clamp + tiny σ°_oh) → Dobson-mineral **−17.92 dB** (anchor −11, Δ = 6.92 dB).

The Phase E-1 mechanism decomposition additionally showed that two-way canopy extinction at Set A θ = 30° is only 8.68 dB, not the ≥30 dB that Memo 2's extinction-saturation hypothesis required. The flat Set A σ° signature is produced by the peat-Mironov clamp suppressing the ground path and leaving the canopy-direct term to dominate, not by canopy extinction saturating the canopy-direct term.

## Finding

The Sets A / B / C published_table failures at the Phase 1b G2 first run (Session D close, 2026-04-18) are not primarily canopy-extinction saturation as Memo 2 hypothesised. They are the same dielectric-configuration category error resolved for the gradient arm by DEV-1b-004, manifesting in the published_table arm.

`g2_anchor_spec.md` v0.3 amended only the E.1 / E.2 gradient rows to specify T94's wheat-reference dielectric. The published_table rows in Sets A (T94 Fig. 2 CHH wheat multiangle), B (T94 Fig. 2 CVV wheat multiangle), and C (T94 Fig. 7 CHH wheat mechanism decomposition) inherit the same implicit assumption — they were evaluated under the Moor House production peat-Mironov configuration in the Session D harness — and require the same amendment. T94's Sets A / B / C rows were generated by T94 using the MIMICS forward model with T94's own soil dielectric, which per T94 §II.A inherits from MIMICS [Ulaby et al. 1990 ref 19] = Dobson 1985 with T94 Table I mineral-soil parameters (verified via text extraction of T94 §II.A, 2026-04-19). Evaluating T94's anchor rows under the Moor House peat parameterisation is a category error: the two configurations are different physical systems (mineral soil at T94's wheat field vs Sphagnum-dominated peat at Moor House blanket bog).

The amendment is scientifically parallel to DEV-1b-004: making previously-implicit configuration explicit per anchor, additive to the pre-registration record, does not modify any anchor value, does not loosen any tolerance, does not remove the DEV-007 clamp, and does not change the Moor House production path.

### Per-row residual classification (for Phase E-2 per-row authorisation)

Using the classification boundaries from the Phase E re-scope decision (< 1 dB = pass; 1–3 dB with canopy-implicated = UMF; 1–3 dB with ground-implicated = Oh / Fresnel; > 3 dB = pause and escalate):

- **Residual < 1 dB:** none yet. The closest is A.2 at 1.96 dB (HH-proxy).
- **Residual 1–3 dB, ground-implicated:** A.2 (1.96), B.2 (1.91). These point at **Oh 1992 vs T94's physical-optics surface scattering** — T94 Table I specifies the physical-optics surface model with s = 0.55 cm, l_s = 4.9 cm (site 13), ks ≈ 0.65, which is an independent surface-model choice our harness did not inherit. This is a **second implicit configuration assumption** analogous to the dielectric one and is discussed in §"Secondary finding" below.
- **Residual 1–3 dB, canopy-implicated:** none in the current preview.
- **Residual > 3 dB:** A.1 (2.55 HH-proxy, climbs over 3 dB when classified honestly vs HH), A.3 (3.98), A.4 (6.54 Brewster), B.1 (3.06), B.3 (3.53), B.4 (7.97 Brewster), C.1 (6.92), C.2 (11.16 — crown mechanism, not dielectric), C.4 (8.15), C.6 (3.27).

The majority of residuals exceed 3 dB and therefore trigger the re-scope-decision's "pause and escalate" clause. Phase E-2 scope authorisation is deferred until the checkpoint report classifies each row with Phase E-1b's final post-amendment numbers and implicates specific mechanisms. The Phase E-2 scope will likely mix three promotion directions: **Oh-vs-PO surface scattering** (Sets A / B / C.1 ground-direct rows), **UMF finite-cylinder form factors** for C.2 specifically (crown-direct saturation at this crop-scale wheat canopy — the Memo 2 hypothesis reduced to a single-row scope), and **literal Fresnel Γ** for the A.4 / B.4 Brewster region (VV-specific, θ=50°, where √σ°_oh as the reflectance proxy miscalibrates the sign of the VV Brewster drop).

### Secondary finding (not resolved in DEV-1b-007)

The Phase E-1b amendment addresses the **dielectric** configuration category error. The preview also surfaces a **surface-scattering** configuration category error: T94's anchor rows were generated with the **physical-optics** surface model (T94 §II.A, Table I), whereas Phase 1b production and the G2 harness use **Oh 1992** (SPEC §7). This is structurally analogous to the dielectric finding and may be partially responsible for the ground-implicated 1–3 dB residuals after Phase E-1b.

**This finding is NOT resolved in DEV-1b-007.** Surface-scattering model choice is a load-bearing Phase 1b scientific decision: SPEC §7 requires the Oh ks-validity check (passed in G3) and the s = 1–5 cm learnable range designed around Oh. Swapping Oh for physical optics (or making it harness-selectable analogous to the dielectric amendment) has larger scope impact than the dielectric swap and requires separate science-agent sign-off. DEV-1b-007 surfaces the finding and recommends it as a Phase E-2 item (per-row where ground-direct is the dominant mechanism and residual is > 1 dB post-Phase-E-1b) **or** a Session F diagnostic-only thread, pending science-agent scope decision.

## Resolution

1. Amend `g2_anchor_spec.md` to v0.4. Add a dielectric-configuration block to Sets A, B, and C analogous to the v0.3 block on Set E. Specifically: Sets A, B, C are evaluated against PyTorch MIMICS configured with `ground_epsilon_dobson_torch` using `eps_dry = DOBSON_EPS_DRY_MINERAL = 3.0`, `eps_water = DOBSON_EPS_WATER = 80.0`, `alpha = DOBSON_ALPHA_MINERAL = 0.65` — Dobson 1985's generic moderate-clay-fraction mineral loam at T94's Table I wheat-field m_v = 0.17 g/cm³.

2. Update `phase1b/physics/equivalence_check.py`. Three call sites:
   - `_run_set_A_B` passes `ground_dielectric_fn = _t94_mineral_dobson` to `mimics_toure_single_crown_breakdown_torch` for Set A (HH) and Set B (VV).
   - `_run_set_C` passes the same callable to `mimics_toure_single_crown_breakdown_torch`.
   - `_run_set_A_B` **proxy-choice correction (companion to this DEV):** mimics.py v0.1 exposes (VV, VH) only. For Set A rows (pol=="HH"), Session D's harness used VH as the HH proxy. VH is cross-pol and is typically 5–8 dB below HH at C-band wheat configurations — a systematic distortion of the Set A residual characterisation. DEV-1b-007's update switches the Set A HH proxy to **VV** (co-pol, typically within ~1 dB of HH at these operating points). Both `sigma_torch_vv_db` and `sigma_torch_vh_db` are exposed in the result JSON row for transparency. Real HH as a first-class channel is a Phase E-2 P5 deliverable and resolves the proxy choice entirely.

3. Moor House production path is unchanged. The existing regression test from DEV-1b-004 (`TestMoorHouseProductionPinning` in `tests/unit/test_mimics_torch.py`) continues to enforce that the PINN-MIMICS trainer and inference paths use `ground_dielectric_fn=None` → SPEC §6 Mironov.

4. Set C2 (U&L 2014 Table 11-1 secondary mechanism ratios) retains its v0.3 `status: DEFERRED_PHASE_E2` marker; Set C2 anchors are structurally Rayleigh-equivalent idealised scenarios and require post-promotion mechanism decomposition to score meaningfully.

5. Set D (M90 walnut L-band) retains its v0.3 `status: EXEMPT` marker per DEV-1b-005.

## Not loosening tolerance

G2 published_table tolerance (±0.5 dB on Sets A / B / C; ±2.0 dB on Set C.5) is unchanged. The pre-registered gate condition — PyTorch MIMICS σ° within tolerance of the anchor value — is unchanged. What is amended is the dielectric configuration under which the PyTorch forward is invoked.

## Not re-reading anchors

T94 σ° values at Sets A, B, C are unchanged. `phase1b/refs/anchor_reads/anchor_reads_v1.json` is unchanged. Sets A / B / C anchor rows in `g2_anchor_spec.md` v0.4 have identical dB values to v0.3 / v0.2.

## Not removing the clamp

The SPEC §6 / DEV-007 Mironov ε ≥ 1.01 clamp remains as pre-registered in the Moor House production path. The amendment routes Sets A / B / C through a **different dielectric function** (Dobson with mineral kwargs) whose `.clamp(min=EPSILON_GROUND_MIN)` is preserved for parity but is inactive at mineral-soil operating points.

## Gate impact

**G2, published_table arm.** After the amendment, re-run G2 and produce per-row residual classification for Phase E-2 scope authorisation. The published_table arm may or may not pass in Phase E-1b — the checkpoint report records the verdict. If some rows still fail after the amendment, the re-scope decision's per-row authorisation tree applies.

**G1 / G3 / G4:** no impact. The amendment touches only the G2 harness call pattern for Sets A / B / C; no Phase 1 baseline, no Oh ks-validity evaluation path, no Dobson-vs-Mironov diagnostic is affected.

**Gradient arm (Set E):** no impact beyond what DEV-1b-004 already established. E.1 / E.2 continue to use Dobson-mineral (DEV-1b-004); E.3 / E.4 / E.5 continue to use the default Mironov path.

## Scope meta-observation

DEV-1b-004 and DEV-1b-007 together constitute the complete resolution of the dielectric-configuration category error discovered during Session E. The anchor spec v0.4 now specifies, for every G2 row that is scored against pass/fail, which dielectric configuration the PyTorch forward is invoked under. This was previously implicit across v0.1 / v0.2 / v0.3. G2 tests the MIMICS physics chain in the reference-paper configurations (T94 for Sets A / B / C / E; M90 for the currently-exempt Set D); Phase 1b training tests the Moor House production configuration. Making this distinction explicit across the whole anchor set sharpens the pre-registration rather than weakening it.
