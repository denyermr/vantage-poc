# DEV-1b-004 — G2 gradient-arm dielectric-configuration amendment (E.1 / E.2)

**Date:** 2026-04-19
**Gate:** G2 (MIMICS Implementation Gate, gradient spot-check arm)
**Session:** Phase 1b, Session E, Phase E-1
**Sibling DEV entries:** DEV-1b-005 (Set D Phase 1c exemption), DEV-1b-006 (v0.2 physics promotion)
**Cross-references:** Decision Memo 1; SPEC.md §6, §11 Diagnostic C; `phase1b/physics/g2_anchor_spec.md` v0.2 → v0.3; Phase 1 DEV-007 (ε ≥ 1.01 Mironov clamp).

---

## Spec said

`SPEC.md` §4 and `phase1b/physics/g2_anchor_spec.md` v0.2 §Set E specify the G2 gradient spot-check arm against Toure 1994 Table V(a) sensitivity coefficients evaluated at the T94 Table IV(a) wheat reference cover (m_v = 0.2 g/cm³, θ = 30°, L-band):

> All five rows (E.1–E.5) pass individually (PyTorch vs T94 tolerance).
> … PyTorch value within ±20% of T94 value, or ±0.1 dB, whichever is larger.

The anchor spec v0.2 did not specify which **dielectric parameterisation** the PyTorch module should use when evaluating these rows. The harness in Session D inherited the SPEC §6 primary Mironov GRMDM with the DEV-007 ε ≥ 1.01 clamp — the Moor House production configuration.

## What was done

At the first G2 run (2026-04-18, end of Session D), rows E.1 (∂σ°_VV/∂m_v) and E.2 (∂σ°_HH/∂m_v) returned `autograd = 0.000 dB`, `finite_difference = 0.000 dB` against T94 Table V(a) sensitivities of 1.21 dB and 1.16 dB. Both autograd and FD disagree with T94 by the full anchor value.

Direct probe of `ground_epsilon_mironov_torch` at T94 Table IV(a) wheat reference m_v = 0.2 g/cm³ under the peat parameterisation (sand = 0%, clay = 0%, organic = 100% — the SPEC §6 primary for Moor House):

```
m_v = 0.10  →  ε = 1.0100,  ∂ε/∂m_v = 0.000   (clamped)
m_v = 0.20  →  ε = 1.0100,  ∂ε/∂m_v = 0.000   ← E.1 / E.2 reference
m_v = 0.30  →  ε = 1.0100,  ∂ε/∂m_v = 0.000   (clamped)
m_v = 0.50  →  ε = 1.0100,  ∂ε/∂m_v = 0.000   (clamped)
m_v = 0.60  →  ε = 1.0750,  ∂ε/∂m_v = 1.855   (unclamped)
m_v = 0.80  →  ε = 1.4757,  ∂ε/∂m_v = 2.162   (unclamped)
```

The peat-Mironov parameterisation produces ε < 1.01 for all m_v ≲ 0.55, so the DEV-007 ε ≥ 1.01 clamp is active at the T94 wheat reference point and nullifies the gradient through the ε path by construction.

Session E amends rows E.1 and E.2 of `g2_anchor_spec.md` to v0.3 with an explicit dielectric-configuration block: E.1 and E.2 are evaluated against PyTorch MIMICS configured with **Dobson 1985 mineral-soil parameters** (the dielectric T94 itself uses; T94 inherits the soil dielectric from MIMICS [Ulaby 1990 ref 19] which uses Dobson 1985, and T94 does not override — verified via text extraction of T94 §II.A and surrounding sections on 2026-04-19). The Moor House production path is unchanged and continues to use the SPEC §6 primary Mironov GRMDM with the DEV-007 clamp.

Code changes:

1. `phase1b/physics/mimics.py` — added `ground_epsilon_dobson_torch(m_v, eps_dry, eps_water, alpha)` as a parameterised differentiable Dobson implementation. Defaults are the Moor House peat values (ε_dry = 3.5, ε_water = 80.0, α = 1.4) mirroring the frozen `phase1/physics/dielectric.DobsonDielectric`. The G2 harness calls it with `eps_dry = DOBSON_EPS_DRY_MINERAL = 3.0` and `alpha = DOBSON_ALPHA_MINERAL = 0.65` (Dobson 1985's generic moderate-clay-fraction mineral loam) for E.1 / E.2 only.

2. `phase1b/physics/mimics.py` — `mimics_toure_single_crown` accepts a keyword-only `ground_dielectric_fn: Optional[Callable[[torch.Tensor], torch.Tensor]]` argument. `None` (default, production) uses `ground_epsilon_mironov_torch` as before; G2's Set E harness passes `ground_epsilon_dobson_torch(…, mineral kwargs)`. A regression test in `tests/unit/test_mimics_torch.py` pins the production path to `ground_dielectric_fn=None`.

3. `phase1b/physics/equivalence_check.py` — Set E gradient-arm evaluation passes the Dobson mineral-soil callable for E.1 / E.2; E.3, E.4, E.5 continue to use the default (Mironov) because they perturb stem height and leaf width, not m_v.

4. `phase1b/physics/g2_anchor_spec.md` — amended to v0.3. §Set E now contains an explicit dielectric-configuration block for E.1 / E.2 (Dobson mineral parameters, inherited from T94 via MIMICS [Ulaby 1990]). Anchor values (1.21 dB, 1.16 dB) **unchanged**.

## Reason

The anchor-spec v0.2 did not make the dielectric parameterisation explicit. The implicit interpretation — "use the Moor House production configuration" — is a category error: T94 Table V(a)'s sensitivity coefficients were derived from T94's wheat-field mineral-soil Dobson configuration, not from a blanket-bog peat configuration. Evaluating the anchor under a physically different substrate is not a test of the MIMICS chain; it is a test of how the clamp behaves at a random operating point.

Making the configuration-of-test explicit per anchor is the scientifically correct response to the first-run finding. It is **additive** to the pre-registration record (an implicit specification becomes explicit) and **does not** modify any anchor value or loosen any tolerance.

## Gate impact

**G2, gradient spot-check arm.** This amendment is specific to rows E.1 and E.2. Rows E.3, E.4, E.5 are unaffected (they perturb stem height and leaf width, and use Mironov as before — Mironov's clamp does not drive those gradients to zero).

**G1:** no impact. G1 covers Phase 1 baseline reproducibility; this DEV is Phase 1b G2-internal.
**G3:** no impact. G3 tests Oh over s = 1–5 cm at Moor House peat; this DEV does not touch that path.
**G4:** no impact. G4 is the Dobson vs Mironov diagnostic over the Moor House m_v range under the peat configuration; this DEV does not modify that diagnostic. (Note: G4 already measured a 97.6 % max relative |Δε|/ε between Dobson-peat and Mironov-peat at Moor House, recorded as "binding = YES". DEV-1b-004's use of Dobson mineral for E.1 / E.2 only does not affect that finding.)

## Resolution

Amendment to `g2_anchor_spec.md` v0.3. Code changes in `phase1b/physics/mimics.py` and `phase1b/physics/equivalence_check.py` per above. Regression test added in `tests/unit/test_mimics_torch.py` to pin the production path to `ground_dielectric_fn=None`. Re-run G2 at Phase E-1 close; expected: E.1 and E.2 now pass within the ±20 % / ±0.1 dB tolerance of T94 Table V(a).

## Sibling observation

The Set A published_table failure (PyTorch σ°_CHH ≈ −20.7 dB flat vs anchors −9.48 to −12.70 dB) and Set C's ground-direct mechanism collapse to −64.6 dB are plausibly the same dielectric-configuration interaction propagating through the published_table arm: at T94's wheat m_v = 0.2 with the peat-Mironov dielectric, the ground-direct path is suppressed to ε = 1.01 regardless of true soil moisture. DEV-1b-006 (Session E Phase E-2) addresses the published_table and mechanism arms via the v0.2 physics promotion (UMF finite-cylinder form factors). The two resolutions are complementary and structurally similar: both rest on the observation that the anchor set inherited implicit configuration assumptions that the first G2 run made explicit, and both make the configuration-of-test explicit per anchor.
